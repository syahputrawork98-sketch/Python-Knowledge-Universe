# SR-07: Modules & Packages (Project Organization)

> **"A project is not just code; it is a structured universe of modules and packages."**

Sub-Rak ini mengeksplorasi bagaimana mengorganisir kode Python dari tingkat file tunggal hingga struktur paket skala besar. Kita akan membedah mekanisme sistem import, cara kerja `sys.path`, serta standar modern distribusi proyek menggunakan `pyproject.toml`.

---

## 🌍 Landscape (Daftar Buku)

| Buku | Fokus | Deskripsi |
| :--- | :--- | :--- |
| **BK-01_Foundations_Modules** | Units | File tunggal, `__main__`, dan mekanisme import dasar. |
| **BK-02_Packages_Organization** | Composite | Struktur folder paket, `__init__`, dan navigasi import. |
| **BK-03_ModernPackaging** | Distribution | Standar metadata `pyproject.toml` dan SRC layout. |

---

## 🎯 Key Learning Goals
- Memahami alur resolusi **Import Search Order** via `sys.path`.
- Menguasai teknik pengorganisasian paket modular untuk proyek skala besar.
- Mampu menyusun metadata proyek standar industri menggunakan **PEP 517/518**.
