# SR-01: Environment Setup (The Serpent's Nest)

> **"A clean workspace is the first step to bug-free code."**

Sub-Rak ini berfokus pada persiapan infrastruktur sebelum menulis baris kode Python pertama. Kita akan membedah bagaimana CPython diinstal, bagaimana mengelola beberapa versi Python, dan pentingnya isolasi dependensi melalui Virtual Environments.

---

## 🌍 Landscape (Daftar Buku)

| Buku | Fokus | Deskripsi |
| :--- | :--- | :--- |
| **BK-01_PythonInterpreters** | Instalasi & Runtime | CPython, Installation, & CLI basics. |
| **BK-02_VirtualEnvironments** | Isolasi Proyek | Konsep isolasi menggunakan modul `venv`. |
| **BK-03_PackageManagement** | Manajemen Paket | `pip`, PyPI, dan standar `pyproject.toml`. |

---

## 🎯 Key Learning Goals
- Mampu menginstal dan mengelola Python menggunakan `pyenv` atau installer resmi.
- Memahami perbedaan antara *Global Python* dan *Isolasi venv*.
- Mampu mengelola paket pihak ketiga secara disiplin menggunakan `pip`.

---

## 🧪 Prasyarat Teknis
- Akses terminal/shell (PowerShell/Bash).
- Koneksi internet untuk mengunduh interpreter dan paket.
