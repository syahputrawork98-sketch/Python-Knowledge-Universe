# SR-03: Collections & Structures (The Data Aggregates)

> **"Data is only as useful as the structure that holds it."**

Sub-Rak ini mengeksplorasi koleksi data bawaan Python yang sangat fleksibel. Kita akan membedah bagaimana Python menyimpan daftar (List), pasangan kunci-nilai (Dict), dan himpunan (Set) di dalam memori, serta kapan harus memilih satu struktur di atas yang lain berdasarkan performa.

---

| **BK-02_Mappings_Sets** | Hashed & Unique | `dict`, `set`, dan pencarian cepat O(1). |

---

## 🎯 Key Learning Goals
- Memahami konsep **Mutability**: Objek mana yang bisa diubah dan mana yang tetap.
- Mampu membedah **Time Complexity (Big O)** untuk operasi dasar (search, insert, delete).
- Memahami mekanisme **Hashing** di balik efisiensi Dictionary dan Set.
