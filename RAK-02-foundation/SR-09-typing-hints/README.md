# SR-09: Typing Hints (Static Semantics)

> **"Explicit typing is better than implicit guessing."**

Sub-Rak ini mengeksplorasi **Modern Python Foundation** melalui sistem pemberian petunjuk tipe (*Type Hints*). Sejak PEP 484, Python berpindah dari bahasa yang murni dinamis menjadi bahasa yang mendukung *Static Type Checking* untuk meningkatkan keandalan kode di skala besarr.

---

## 🌍 Landscape (Daftar Buku)

| Buku | Fokus | Deskripsi |
| :--- | :--- | :--- |
| **BK-01_BasicAnnotations** | Dasar Typing | Mengetahui `Optional`, `Any`, `List`, `Dict`. |
| **BK-02_TypeSafetyMypy** | Verifikasi Statis | Menggunakan alat bantu seperti Mypy/Pyright. |

---

## 🎯 Key Learning Goals
- Memahami beda antara *Dynamic Typing* (Runtime) dan *Static Type Checking*.
- Mampu membubuhkan *Type Annotations* pada variabel, parameter, dan hasil fungsi.
- Mengetahui penggunaan *Type Hinting* sebagai dokumentasi aktif.

---

## 🧪 Prasyarat Teknis
- Pengetahuan dasar fungsi dan variabel di Python.
- Instalasi `mypy` (opsional namun disarankan).
