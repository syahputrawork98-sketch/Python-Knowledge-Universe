# SR-02: Data Types (The Primitives) [x] Complete

> **"Data is the sub-layer of reality; data types are the rules of that reality."**

Sub-Rak ini membedah tipe data dasar yang menjadi blok pembangun utama dalam Python. Kita akan mempelajari bagaimana Python menangani angka, teks, dan logika boolean sebagai objek, serta memahami konsep penting **Immutability** yang membedakan tipe data primitif dari struktur data kompleks.

## 🎯 Key Learning Goals
- Memahami konsep **Everything is an Object** di Python.
- Menguasai perbedaan antara **Immutable** (Primitif) dan **Mutable** (Koleksi).
- Memahami representasi internal tipe data numerik dan teks di memori CPython.
- Mengenal singleton `None` sebagai representasi ketiadaan nilai.
