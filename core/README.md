# Rak Utama: Core Python

Rak ini adalah fondasi. Anda tidak perlu menguasai semuanya sebelum mulai membangun aplikasi nyata, namun menyempurnakan ilmu di rak ini akan membuat pemahaman fundamental Python Anda tak tergoyahkan.

## Susunan Sub-Rak Pembelajaran:

1. **[01_fundamentals](01_fundamentals/)** - Mulai dari sini. Belajar sintaks, logika data, paradigma OOP, dan desain bahasa.
2. **[02_advanced_concepts](02_advanced_concepts/)** - Pahami apa yang terjadi pada *mesin* Python: Model Eksekusi dan Memori.
3. **[03_system_and_io](03_system_and_io/)** - Komunikasi dengan dunia luar: Sistem Operasi, IO, dan Modul.
4. **[04_concurrency](04_concurrency/)** - Program yang melakukan beberapa hal secara paralel.
5. **[05_ecosystem_and_tooling](05_ecosystem_and_tooling/)** - Lingkungan dan perkakas pengembangan profesional.
