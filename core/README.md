# Core Python

Bagian ini berisi **peta pengetahuan inti (Core Knowledge) dari bahasa pemrograman Python**.

Materi pada bagian ini membahas konsep-konsep fundamental Python yang menjadi dasar sebelum mempelajari bidang spesialisasi seperti web development, data science, machine learning, AI engineering, maupun automation.

Struktur pada bagian Core Python disusun sebagai **knowledge map**, sehingga setiap topik penting dalam Python memiliki tempat yang jelas dan terorganisir. Pendekatan ini membantu pembaca memahami Python secara lebih sistematis, mulai dari dasar bahasa hingga ekosistem pengembangannya.

Urutan topik juga dirancang sebagai **jalur belajar yang direkomendasikan**, sehingga pembaca pemula dapat mengikuti materi secara bertahap tanpa kebingungan.
