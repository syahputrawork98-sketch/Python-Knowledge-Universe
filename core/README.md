# Core Python

Bagian **Core Python** berisi fondasi utama bahasa Python.

Sebelum mempelajari bidang spesialisasi seperti web development, data science, atau machine learning, penting untuk memahami konsep dasar Python secara menyeluruh.

Core Python dibagi menjadi **7 buku utama**.

---

# Struktur Core

| No | Buku                  | Fokus                        |
| -- | --------------------- | ---------------------------- |
| 01 | Tutorial              | Sintaks dasar Python         |
| 02 | Language Principles   | Konsep dasar bahasa          |
| 03 | Runtime Execution     | Cara Python menjalankan kode |
| 04 | Object & Memory       | Model objek Python           |
| 05 | Standard Library & IO | Library bawaan               |
| 06 | Concurrency & Async   | Paralelisme                  |
| 07 | Packaging & Ecosystem | Distribusi Python            |

---

# Tujuan Core Python

Setelah menyelesaikan bagian ini, pembaca diharapkan:

* memahami sintaks Python
* memahami model objek Python
* memahami cara kerja interpreter Python
* mampu menulis program Python yang terstruktur
* siap masuk ke bidang spesialisasi
