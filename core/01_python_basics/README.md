# Python Basics

Book Code: CORE-01
Version: v0.2.1
Last Updated: 2026-03-08
Status: In Progress

Bagian ini membahas fondasi dasar bahasa pemrograman Python.

Materi pada bagian ini dirancang untuk membantu pembaca memahami konsep dasar Python sebelum mempelajari topik lanjutan dalam Core Python.

## Tujuan Bagian Ini

- memahami sintaks dasar Python
- mengenal tipe data dan struktur data dasar
- memahami kontrol alur program
- memahami cara mendefinisikan dan menggunakan fungsi
- memahami konsep dasar modul dan import
- memahami cara melakukan input/output
- memahami mekanisme error dan exception
