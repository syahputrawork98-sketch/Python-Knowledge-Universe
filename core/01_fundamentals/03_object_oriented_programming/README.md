# Object Oriented Programming

Book Code: CORE-03
Version: Core.Fundamentals.03.00
Last Updated: 2026-03-14
Status: Planned

Buku ini membangun fondasi paradigma pemrograman berbasis objek yang sangat kuat, bertindak sebagai jembatan sebelum masuk ke materi internal objek di Rak 02.

## Tujuan Bagian Ini

- memahami filosofi dan kebutuhan akan OOP
- menguasai pembuatan Class dan Object
- memahami peran `__init__` dan `self`
- membedakan berbagai jenis method (instance, class, static)
- menguasai konsep pewarisan (Inheritance)
- memahami enkapsulasi dan akses modifier
- menguasai polimorfisme dan abstraksi (ABC)
- memahami komposisi vs pewarisan
- menggunakan Property Decorators secara efektif
