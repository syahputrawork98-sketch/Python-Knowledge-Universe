# Data and Logic

Book Code: CORE-02
Version: Core.Fundamentals.02.00
Last Updated: 2026-03-14
Status: Published

Buku ini berfokus pada cara mengorganisir data dan alur logika program yang lebih kompleks, mulai dari fungsi hingga pola pemrograman dasar.

## Tujuan Bagian Ini

- memahami penggunaan fungsi secara mendalam (parameter, scope)
- mengenal fungsi anonim (lambda)
- menguasai struktur data bawaan Python (List, Tuple, Set, Dict)
- memahami manipulasi string tingkat lanjut
- memahami modularitas melalui modul dan paket
- melakukan operasi I/O dan manajemen file
- menangani error melalui exception
- menerapkan pola pemrograman dasar (Accumulator, Filtering, dll)
