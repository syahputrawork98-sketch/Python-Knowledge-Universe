# Roadmap

Dokumen ini memetakan rencana pengembangan Python Knowledge Universe.

## Fase Pengembangan

### Phase 1: Foundation (In Progress)

- finalisasi struktur repository
- penyelarasan penamaan topik
- pembuatan README inti untuk root, core, dan specialization

### Phase 2: Core Authoring (Planned)

- penulisan konten `01_python_basics`
- penulisan konten 14 topik core lainnya
- validasi referensi ke dokumentasi resmi Python

### Phase 3: Specialization Authoring (Planned)

- desain kurikulum per specialization
- pembuatan konten Web Development
- pembuatan konten Data Science
- pembuatan konten Machine Learning
- pembuatan konten AI Engineering
- pembuatan konten Automation

### Phase 4: Quality and Publishing (Planned)

- standardisasi template dokumen
- review teknis dan editorial
- publikasi bertahap ke website dokumentasi

## Milestone

- M1: Struktur core dan specialization stabil
- M2: Python Basics selesai ditulis
- M3: Core Python selesai versi pertama
- M4: Minimal 2 specialization tersedia
- M5: Website dokumentasi versi awal rilis
