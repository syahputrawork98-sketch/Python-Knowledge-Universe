# Glossary

Dokumen ini berisi istilah penting yang digunakan di Python Knowledge Universe.

## Istilah Umum

- `Core Python`: Fondasi konsep Python yang berlaku lintas domain.
- `Specialization`: Jalur penerapan Python pada domain tertentu.
- `Execution Model`: Mekanisme Python menjalankan kode.
- `Object Model`: Aturan representasi objek dalam Python.
- `Memory Management`: Pengelolaan alokasi dan pelepasan memori.
- `Module`: Unit kode Python yang dapat diimpor.
- `Package`: Kumpulan module dalam satu struktur distribusi.
- `Virtual Environment`: Lingkungan Python terisolasi untuk dependency.
- `Tooling`: Alat bantu pengembangan seperti formatter, linter, dan test runner.
- `PEP`: Python Enhancement Proposal, dokumen standar evolusi Python.

## Status Konten

- `Planned`: Topik sudah didefinisikan, konten belum ditulis.
- `In Progress`: Konten sedang ditulis atau direview.
- `Published`: Konten siap digunakan.
- `Revision`: Konten dipublikasikan tetapi butuh pembaruan.
