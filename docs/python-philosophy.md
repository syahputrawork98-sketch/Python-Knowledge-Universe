# Filosofi & Esensi Python

> **"Readability Counts. Beautiful is better than ugly."**

## Mengapa Python Ada?

Python hadir dengan misi untuk **"Memanusiakan Kode"**. Filosofi utamanya adalah bahwa kode software lebih sering dibaca daripada ditulis. Python dirancang agar sintaksisnya terasa seperti bahasa Inggris yang bersih, menghilangkan hambatan visual (seperti kurung kurawal atau titik koma) agar pengembang bisa fokus pada penyelesaian masalah, bukan pada aturan bahasa.

## Konsep "The Zen of Python" (PEP 20) 🐍

Filosofi Python dirangkum secara elegan dalam 19 aturan emas, di antaranya:

### 1. Eksplisit Lebih Baik daripada Implisit
Python tidak suka keajaiban di balik layar yang membingungkan. "Jangan sembunyikan niat Anda." Jika sesuatu harus dilakukan, lakukanlah secara jelas.

### 2. Hanya Ada Satu Cara Jelas (The Zen)
Berbeda dengan bahasa seperti Perl atau Ruby yang memiliki filosofi "banyak cara untuk melakukannya", Python lebih suka ada satu (dan sebaiknya hanya satu) cara yang jelas dan idiomatik untuk menyelesaikan tugas. Ini meminimalkan kebingungan saat membaca kode orang lain.

### 3. Batteries Included
Filosofi "Batteries Included" berarti Python datang dengan pustaka standar yang sangat kaya. Anda bisa membangun web server, mengolah data, atau mengirim email tanpa harus menginstal pustaka pihak ketiga.
