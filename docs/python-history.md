# Sejarah & Evolusi Python

> **"From a Christmas Hobby to the King of Data Science."**

## 1. Kelahiran (1989 - 1991)

Python diciptakan oleh **Guido van Rossum** pada akhir Desember 1989 di CWI (Centrum Wiskunde & Informatica), Belanda. Proyek ini dimulai sebagai hobi selama liburan Natal untuk mengisi waktu luang. Nama **Python** sendiri diambil dari acara komedi BBC favoritnya, *Monty Python's Flying Circus*.

### Latar Belakang Masalah
Guido ingin menciptakan penerus bahasa **ABC** yang mampu menangani pengecualian (exceptions) dan berinteraksi dengan sistem operasi Amoeba, namun dengan sintaksis yang jauh lebih bersih dan keterbacaan tinggi.

## 2. Era Pertumbuhan & Python 2.0 (1994 - 2000)

Python merilis versi 1.0 pada tahun 1994 dengan fitur *functional programming* (lambda, map, filter).
- **Python 2.0 (2000)**: Memperkenalkan fitur krusial seperti *list comprehensions* dan sistem *garbage collection* berbasis *reference counting*. Ini adalah era di mana Python mulai populer di kalangan sysadmin dan pengembang web (Zope/Plone).

## 3. Krisis Transisi: Python 3.0 (2008)

Python 3.0 dirilis dengan perubahan radikal yang tidak kompatibel ke belakang (*backward-incompatible*). Ini adalah keputusan besar untuk memperbaiki kesalahan desain fundamental bahasa.
- **Masalah Unicode**: Python 3 memperbaiki penanganan string agar mendukung Unicode secara default.
- **Era Dualisme**: Selama hampir satu dekade, komunitas terbelah antara Python 2 dan Python 3, sebelum akhirnya Python 2 resmi dihentikan (*sunset*) pada 1 Januari 2020.

## 4. Ledakan AI & Data Science (2015 - Sekarang)

Python mengalami lonjakan popularitas luar biasa berkat ekosistem pustaka matematika dan data yang masif:
- **Pilar Utama**: NumPy, Pandas, Scikit-Learn, TensorFlow, dan PyTorch menjadikan Python sebagai bahasa standar untuk AI and Machine Learning.
- **Modern Python**: Versi terbaru (3.10+) terus berevolusi dengan fitur seperti *Structural Pattern Matching* dan peningkatan performa yang signifikan.
