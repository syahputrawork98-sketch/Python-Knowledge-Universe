# Rak 14: Automation & Scripting

Implementasi Python sebagai "Glue Language" untuk mengotomatisasi sistem dan alur kerja digital.

## Struktur Sub-Rak
- **[SR-01-SYSTEM-AUTOMATION](./SR-01-SYSTEM-AUTOMATION/README.md)**: Scripting tingkat OS dan file system.
- **[SR-02-WEB-AUTOMATION](./SR-02-WEB-AUTOMATION/README.md)**: Browser automation & Web Scraping (Playwright).

## Why Python for Automation?
- **Standard Library**: Modul `os`, `shutil`, dan `pathlib` sangat intuitif.
- **Speed of Development**: Menulis skrip otomasi 10 baris di Python setara dengan 50 baris di bahasa lain.
- **Cross-Platform**: Skrip yang sama berjalan stabil di Windows, Linux, dan macOS.
