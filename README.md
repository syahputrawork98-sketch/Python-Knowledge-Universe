# Python Knowledge Universe

Python Knowledge Universe adalah repositori pembelajaran yang dirancang sebagai peta pengetahuan Python yang terstruktur, mulai dari konsep dasar hingga bidang spesialisasi.

Repositori ini bertujuan membantu developer memahami Python secara mendalam dan sistematis.

Struktur pembelajaran dibagi menjadi dua bagian utama:

* **Core Python Knowledge** – memahami fondasi bahasa Python
* **Python Specializations** – penerapan Python di berbagai bidang

Target pembelajaran repositori ini adalah:

Python Beginner → Python Expert → Domain Specialist

---
