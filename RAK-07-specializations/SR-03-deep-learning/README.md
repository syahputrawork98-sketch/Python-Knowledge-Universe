## 🎯 Key Learning Goals
- Memahami arsitektur **Retrieval-Augmented Generation (RAG)** secara end-to-end.
- Mampu membangun orkestrasi AI menggunakan komponen **Chains**, **Prompts**, dan **Retriever**.
- Memahami konsep **Embedding** dan pencarian semantik pada **Vector Database**.
- Mampu menjalankan model bahasa canggih (Llama, Mistral) secara lokal via **Transformers**.
- Mengenali mekanisme **Self-Attention** yang mendasari kecerdasan model GPT.
