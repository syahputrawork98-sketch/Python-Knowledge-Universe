## 🎯 Key Learning Goals
- Memahami mekanisme **Automatic Differentiation** (Autograd) di level C++/Python.
- Mampu membangun dan melatih model regresi/klasifikasi sederhana menggunakan **PyTorch**.
- Menguasai penggunaan **Device Management** (CPU vs GPU CUDA) untuk akselerasi model.
- Membangun pipeline prapemrosesan data yang aman dari kebocoran (Data Leakage).
- Mampu melacak parameter dan metriks pelatihan secara rapi menggunakan **MLflow**.
