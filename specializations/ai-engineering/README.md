# AI Engineering

Sub-rak ini berfokus pada pembangunan sistem AI berbasis Python untuk use case nyata.

## Peran Sub-rak

- menerjemahkan fondasi Core Python ke sistem AI modern
- menyusun jalur belajar dari LLM dasar sampai integrasi aplikasi AI
- menyediakan buku arsitektur, implementasi, dan project AI end-to-end
