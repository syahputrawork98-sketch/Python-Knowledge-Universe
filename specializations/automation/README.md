# Automation

Sub-rak ini berfokus pada penggunaan Python untuk otomatisasi tugas dan workflow.

## Peran Sub-rak

- menerjemahkan fondasi Core Python ke otomasi operasional
- menyusun jalur belajar dari scripting dasar sampai orkestrasi workflow
- menyediakan buku konsep, praktik otomasi, dan project implementasi
